# AI 基础知识

## 什么是人工智能？

人工智能（Artificial Intelligence，简称 AI）是计算机科学的一个分支，旨在创建能够模拟人类智能的系统。这包括学习、推理、问题解决、感知和语言理解等能力。

## 机器学习

机器学习是 AI 的一个子集，让系统从数据中学习而不需要显式编程。主要类型：

- **监督学习**：使用标注数据训练模型
- **无监督学习**：发现数据中的模式
- **强化学习**：通过奖惩机制学习

## 深度学习

深度学习使用多层神经网络，是机器学习的一个子集。关键概念：

- **神经网络**：由神经元层组成的计算模型
- **CNN**：卷积神经网络，擅长图像处理
- **RNN**：循环神经网络，擅长序列数据
- **Transformer**：基于注意力机制的架构，是现代 LLM 的基础

## 大语言模型（LLM）

大语言模型是使用海量文本训练的大型神经网络。代表模型：

- GPT 系列（OpenAI）
- Claude 系列（Anthropic）
- DeepSeek
- LLaMA 系列（Meta）

## 关键术语

| 术语 | 说明 |
|------|------|
| Token | 文本的最小处理单位 |
| Embedding | 文本的向量表示 |
| 注意力机制 | 让模型关注输入的重要部分 |
| 微调 | 在已有模型上继续训练以适应特定任务 |
| RAG | 检索增强生成，结合搜索与生成 |
